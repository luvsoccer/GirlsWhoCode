Basic Format

Starting the code

To start, the first tag is < !DOCTYPE html > It is one of the only tags that do not have an end tag.

Next you put the < html > tag. Do not forget the end tag which is the same as the beginning except with a back slash in front of it

Then, you put the < head > tag. Always put an end tag except if said not to!

Finally put a < body > tag.




This is how it should look:


< !DOCTYPE html >

< html >

< head >


< title> Text < title>


< /head >


< body >


< /body >

< /html >
